We are happy to announce the next releases of Storm and stormpy in version 1.13.0.
<!--more-->

Storm 1.13.0 adds (among others) support for the (binary) [UMB](https://pmc-tools.github.io/umb/spec) format, support for bounded model checking of interval DTMC/MDP and support for monitor verification.
We refer to the [release notes of Storm](https://github.com/stormchecker/storm/releases/tag/1.13.0) for a complete list of all changes.

The [stormpy](https://stormchecker.github.io/stormpy/) Python bindings are also released in a new version 1.13.0 that comes with support for exact intervals, support for monitor verification and conditional model checking..
We refer to the [release notes of stormpy](https://github.com/stormchecker/stormpy/releases/tag/1.13.0) for all details.

You can get the new release of Storm either by building from [source]({{ '/documentation/obtain-storm/build.html' | relative_url }}), via the pre-built binaries from [Homebrew]({{ '/documentation/obtain-storm/homebrew.html' | relative_url }}), via the pre-built Debian packages attached to the [release](https://github.com/stormchecker/storm/releases/tag/1.13.0) or by using a [Docker container]({{ '/documentation/obtain-storm/docker.html' | relative_url }}).
You can obtain the stormpy Python package by simply executing `pip install stormpy`.

If you experience any bugs, please [let us know]({{ '/documentation/obtain-storm/troubleshooting.html#file-an-issue' | relative_url }}).
